# Infinite Locus — Product Decision Flow Redesign

Senior Product Designer case study by **Mohd Hayaat Ali** · Infinite Locus · 2026.

## Overview

A mobile-first redesign of the **Product Decision Flow** for a DTC fashion brand, focused on the last mile before purchase. The goal is to convert high-intent mobile shoppers into confident buyers by reducing decision friction around fit, size, delivery, and returns.

## Flow Scope

**Selected flow:**  
`PDP → Variant Selection → Add to Bag`

Out of the three assignment flows (Product Discovery, Product Decision, Drop Launch), Product Decision was chosen for highest leverage: it directly impacts Add-to-Cart rate and connects to checkout momentum.

## Key Problem

High-intent mobile shoppers already want the product. They drop off because the PDP does not give enough **confidence before commitment** — size uncertainty, late trust cues, promotional noise, and weak post-ATB confirmation.

## What This Case Study Covers

- **Experience Audit** — Mobile PDP patterns across Indian fashion platforms
- **Design Principles** — Confidence Before Commitment, Keep Users In Context, Make Sizing Guided, Reassurance Near CTA
- **New Concept: Fit Confidence** — A lightweight fit profile + garment data + model reference + buyer feedback system that outputs a size recommendation with a confidence score (e.g., M · 87% match)
- **UX Flow & Wireframes** — 5-screen decision flow from PDP to confirmation
- **Final UI** — High-fidelity screens showing the redesigned PDP, fit discovery sheet, size guide, size selected state, and add-to-bag confirmation
- **Impact & Validation** — Proposed metrics and lightweight test plan for ATB rate, size-related returns, and CSAT
- **Fit Confidence Data Layer** — Separate from the PDP, explains how the 87% score is calculated from weighted signals without invasive data or AI guarantees

## One-Line Summary

Fit Confidence is a decision-support layer that combines user fit profile, garment data, model reference, and buyer feedback into a brand-backed size recommendation — turning mobile PDPs from product pages into confident purchase moments.

## Target Impact

- ↑ Add-to-Bag rate
- ↓ Size-related returns
- ↑ Customer trust and satisfaction
- ↓ Carbon footprint from fewer returns
